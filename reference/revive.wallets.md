# Revive wallets processes

Revive wallets processes

## Usage

``` r
revive.wallets(
  monerod.rpc.port,
  wallets.data.file = "spam_wallets_data.rds",
  restart.alive.processes = FALSE,
  print.json = FALSE,
  cleanup.process.monero_wallet_rpc = FALSE,
  monero.wallet.rpc.log.level = 0
)
```

## Arguments

- monerod.rpc.port:

  monerod.rpc.port

- wallets.data.file:

  wallets.data.file

- restart.alive.processes:

  restart.alive.processes

- print.json:

  print.json

- cleanup.process.monero_wallet_rpc:

  cleanup.process.monero_wallet_rpc

- monero.wallet.rpc.log.level:

  monero.wallet.rpc.log.level

## Value

1
