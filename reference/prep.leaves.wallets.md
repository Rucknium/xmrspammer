# Prepare Wallet Account Tree Leaves

Prepare Wallet Account Tree Leaves

## Usage

``` r
prep.leaves.wallets(
  wallets,
  threads = NA,
  n.outputs = 15,
  n.tree.levels = 4,
  fee.priority = c(3, 3, 3, 0),
  send.txs = TRUE,
  wallets.data.file = "spam_wallets_data.rds"
)
```

## Arguments

- wallets:

  wallets

- threads:

  threads

- n.outputs:

  n.outputs

- n.tree.levels:

  n.tree.levels

- fee.priority:

  fee.priority

- send.txs:

  send.txs

- wallets.data.file:

  wallets.data.file

## Value

1
