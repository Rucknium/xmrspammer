# Generate wallets

Generate wallets

## Usage

``` r
gen.wallets(
  monerod.rpc.port,
  n.wallets,
  wallets.data.file = "spam_wallets_data.rds",
  print.json = TRUE,
  cleanup.process.monero_wallet_rpc = FALSE,
  monero.wallet.rpc.log.level = 0
)
```

## Arguments

- monerod.rpc.port:

  monerod.rpc.port

- n.wallets:

  n.wallets

- wallets.data.file:

  wallets.data.file

- print.json:

  print.json

- cleanup.process.monero_wallet_rpc:

  cleanup.process.monero_wallet_rpc

- monero.wallet.rpc.log.level:

  monero.wallet.rpc.log.level

## Value

1
